        # 供水恢复

        方法：在室内条件下在水龙头上方安置雨水收集桶，再使用管钳右键点击水龙头即可将雨水收集桶与水龙头连接。

        # 我为什么要这么做？

        从水龙头取得水不再需要消毒，水龙头自带过滤效果。接水后可直接饮用或洗涤肮脏的止血道具。

        # 我的水龙头为什么不能连接雨水收集桶

        1. 部分人连接时会使用扳手，但实际需要的工具为管钳
        2. 以水龙头为中心周围范围1格以内的雨水收集桶都是可以连接的。但是不同版本有一些区别：水龙头正上方的位置某些版本会阻碍连接。可在建立多个雨水收集桶之前事先测试一下
        3. 水龙头不在室内范围。（门框，墙壁框架等也属于墙壁，门框不一定装门）
        4. 不能隔楼层接雨水收集桶（1楼的水龙头，3楼的水桶）

        # 当连接的雨水收集桶里的水使用完之后，是不是需要再次连接水龙头？

        不用，当一个雨水收集桶里的水使用完会自动连接下一个可使用的雨水收集器。
