        # 供电恢复

        材料：发电机、发电机手册、油

        方法：首先要使用发电机需要阅读发电机手册（不影响搬运发电机），再将发电机搬运至家中，添加燃油即可恢复已发电机为原点20米半径的电力，高度共7层。（数据来自B站阿婆主 鼠儿尾。想了解干活，找他没错）

        # 在哪里能找到发电机？

        1. 有些棕色居民区自带一些小仓库（不是车库），里面有概率刷新发电机。
        2. 某些黑色大仓库带多个小隔间，小隔间内有概率刷新发电机。

        # 发电机手册在哪里能找到？

        1. 学校图书馆
        2. 棕色居民区的书架上
        3. 邮箱里

        # 发电机耐久如何恢复?

        通过电子废料 + 螺丝刀修理。电工等级越高单次修复的耐久越高。

        # 没有油，又停电了怎么办？

        你不会把发电机搬运到加油站再把车里的油抽了给发电机使用吗？什么？你空手搬运的，没有车？你可真是个猛男！
