# 特性

# 职业

# 失业者：

8点初始特性，让你可以自由发展，建议对游戏熟悉的后选择该职业。

# 消防员：

消防员特性是一个能加速你用斧子砸门、砍僵尸的优质特性，负重增加也很不错，推荐新手选择。

# 警官：

有神射手特性，增加枪械准确度同时提升换弹速度，不过游戏中射击武器实用性有限。

# 公园巡护员：

特性为野外活动者，作用就如其名字，比较实用，可以依靠树林躲避和抄近路。

# 建筑工人：

使用钝器打击的好手，防御能力也不错，战斗向的职业。

# 警卫：

特性是夜猫子，具体就是睡觉时间变短且保持警惕。有更多的活动时间，且过夜时安全性更高。

# 职业选择

# 职业选择

# 木匠：

喜欢制造的玩家可以考虑。

# 窃贼：

喜欢潜入玩法的玩家比较适合。

# 厨师：

多了些食物配方，不过并非很强的能力。

# 修理工：

提高修理物品的能力。

# 农民：

喜欢经营菜地、自给自足的玩家会喜欢这个职业。

# 渔夫：

钓鱼能力强，和农夫可以配合，建立基地。

# 医生：

简单好用，喜欢到处乱跑的家伙尤其需要医生。

# 退伍军人：

战斗能力强，“麻木”。

# 护工：

和医生类似。

# 伐木工：

也是强烈推荐新手的职业，像是消防员和公园巡护员合体。

# 健身教练：

跑得最快的职业。

# 汉堡店员工：

如果选这个，不如选厨师。

# 工程师：

比木匠更全面，能够做炸弹，团队行动时可以选择。

# 电焊工：

可以制作金属物件及盒子，单独选择并没有太大价值。

# 特征

# 清醒

点数消耗：2 额外等级：无 特征冲突：懒鬼 效果：疲劳值消耗效果-33%，奔跑，战斗，工作时，都会消耗人物的疲劳值 建议：疲劳值在41版本变得相当苛刻，如果玩家处于疲劳状态，那么，驾驶和战斗的效果会大打折扣，近战伤害会降低，远程武器精度大幅下降，驾车惩罚等，所以，保持清醒相当重要！ 推荐等级：4

# 超速狂魔

点数消耗：1 额外等级：无 特征冲突：新手司机 效果：飙车时，车速加快100% 建议：这项特征其实更适合电脑屏幕较大的玩家们用，因为这游戏是以上帝视角开车，玩家无法以第一人称的形式观看到路况，所以，开的慢也是件好事，因为游戏中车祸伤害是以写实为主，当然了，如果你多出1点点数，那么就补上这个吧 推荐等级：2【不是很推荐】

# 猫眼

点数消耗：2 额外等级：无 特征冲突：无 效果：晚上时，如果在房屋外，视野会相当明亮，在房屋里时，不会获得夜视效果【天气不好时，如：起雾，下暴雨，夜视效果会大打折扣】 建议：电脑屏幕较暗，或者不喜欢夜晚的黑暗气氛的玩家们可以选择 推荐等级：3【你看着办】

# 读的快

点数消耗：2 额外等级：无 特征冲突：读的慢 效果：正常时间下，挂机读完一本200页的书籍需要10分钟【现实时间】，阅读快则可以把时间变成5分钟读完200页的书籍 建议：加快读书的速度，单人模式大可不必，多人模式一定要点！ 推荐等级： 单人模式：1【浪费点数】 多人模式：5【一定要带】
